### 1. Loop Infinito API (CRITICO - 3 ore)

#### Problema
```
ğŸ“Š Centinaia di richieste al secondo:
GET /api/user/analytics/summary 200 11.637 ms - 340
GET /api/user/analytics/summary 200 11.637 ms - 340
[...continua infinitamente...]

âš ï¸ Error: Maximum update depth exceeded
âš ï¸ Error: Rendered fewer hooks than expected
```

#### Causa Root
Il `getAuthenticatedApiClient` in `useAuth` non era memoizzato, causando:
```typescript
// âŒ PROBLEMA
const getAuthenticatedApiClient = () => {
  if (!token) return null;
  return createAuthenticatedApiClient(token);
}; // â† Ricreato ad ogni render!

// useEffect nei widget
useEffect(() => {
  fetchStats(); // â† Cambia ogni render perchÃ© dipende da getAuthenticatedApiClient
}, [fetchStats]); // â† Loop infinito!
```

#### Soluzione Implementata
```typescript
// âœ… SOLUZIONE
const getAuthenticatedApiClient = useCallback(() => {
  if (!token) return null;
  return createAuthenticatedApiClient(token);
}, [token]); // â† Memoizzato! Cambia solo quando token cambia
```

**File Modificato**: `apps/web/src/hooks/useAuth.ts`
